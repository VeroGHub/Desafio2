Instrucciones de instalacion:


1. Generar imagen Dockerfile_ssh

2. Generar contenedor Ubuntu_ssh, disponibilizando puertos 27017:27017

3. Generar imagen Dockerfile_ansible

4. Generar contenedor Ansible

5. Conectarse a contenedor Ansible y ejecutar el siguiente playbook playdesa2.yml que se encuentra en la ruta /root

6. Al momento de ejecutar playbook este pedira una pass, la pass es "llave"

7. Conectarse a localhost:27017
